Which logo do I use?

Four marks, one hierarchy. When in doubt, use №1 — the gradient main logo. Everything else exists for a specific job; if your situation isn't listed, default to №1 or ask the design team.

1Primary

Main logo — Gradient

  • The default. Anywhere the brand should shine
  • Website, marketing, ads, decks, social
  • Dark version on light bg, light version on brand purple
2Secondary

Main logo — Flat

  • Small sizes, where the gradient gets muddy
  • Limited colours: one-colour print, embroidery, engraving
  • Busy backgrounds where flat reads clearer
  • Not a style preference — a fallback
3Restricted

M letterform

  • Merch: tees, caps, stickers, totes
  • In-app animation: loading states, motion moments
  • Never a logo replacement in marketing or docs
4Restricted

Monogram — M in pill

  • Merch: pins, embroidery, small details
  • Small scale: name cards, tiny print areas
  • Watermarks on photos, docs, video
  • Never the main logo, never the app icon
Quick decision checklist
  1. Marketing, deck, website, anything public-facing? → Gradient main logo. Done.
  2. Small, one-colour, or on a busy background? → Flat main logo.
  3. Merch or an in-app animation? → M letterform (or monogram for merch).
  4. Name card, watermark, or a tiny space? → Monogram.
  5. Still not sure? → Gradient main logo, or ask the design team.

Applies to every mark: never stretch, rotate, recolour, fade, or add effects · respect the clear space · only brand-provided gradients · always use the official files — never retype the logo. The MaxMiles mark represents the currency only, never the brand.
